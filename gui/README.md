# 🖼️ Nuggies GUI (Optional)

The GUI is intentionally **not the primary interface**.

## Philosophy

- Must mirror CLI behavior exactly
- Must not hide system operations
- Must not introduce abstraction layers

## Status

Not implemented.

## Future Ideas

- Minimal GTK or Qt frontend
- Electron is discouraged unless strictly necessary
- Should be forkable and optional

## Rule

> If the GUI does something the CLI cannot explain, it is wrong.
